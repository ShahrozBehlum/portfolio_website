## 📖 About
Modern developer portfolio with cosmic space theme, animated electron orbits, typewriter effects, project showcase, skill metrics, and contact functionality.

## ✨ Features
- 🌟 Animated starfield background
- 🪐 Rotating electron orbit animations
- ⌨️ Dynamic typewriter text
- 📱 Fully responsive design
- 🚀 Live project demos (3 deployed apps)
- 📊 Animated skill progress bars
- 📜 Certificate gallery with links
- 📧 Contact form with email integration

## 🛠️ Tech Stack
- React 18
- Tailwind CSS
- Font Awesome 6
- Web3Forms API
